# Làm việc với Amazon S3, Lambda và API Gateway để tải lên và tải xuống hình ảnh

### Tổng quan

Trong bài lab này, bạn sẽ tìm hiểu các khái niệm cơ bản và thực hành với **Amazon S3, Lambda và API Gateway** để triển khai hệ thống **tải lên và tải xuống hình ảnh**. Bài lab hướng dẫn cách **upload hình ảnh lên S3 và truy xuất lại**, thông qua **API Gateway và AWS Lambda**, và được kiểm thử bằng **Postman**.
