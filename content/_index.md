# Working with Amazon S3, Lambda, and API Gateway for Image Upload and Download

### Overview

In this lab, you will learn the fundamental concepts and hands-on practice with **Amazon S3, Lambda, and API Gateway** to implement an image upload and download system. The lab demonstrates how to **upload an image to an S3 bucket and retrieve it**, using **API Gateway and AWS Lambda**, tested via **Postman**.
